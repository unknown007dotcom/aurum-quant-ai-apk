# XAUUSD Bank Algorithm Analyzer

Local static web app for analyzing live `XAU/USD` market structure with:

- SMA 20 / SMA 50 bias
- FVG detection
- Order Blocks (extreme and decisional)
- BOS and CHoCH
- Midnight line bias
- Unmitigated OB reversal zone detection
- Rule-engine trade decision
- AI explanation through NVIDIA's OpenAI-compatible endpoint

## Run

This build should be run through the local server so the `/api/*` routes are available.

```powershell
node server.js
```

Then open `http://localhost:3000`.
